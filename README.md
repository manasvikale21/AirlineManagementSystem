# ✈️ Airline Management System

## 📌 Description
A console-based Airline Management System built using Java and MySQL.
It allows users to manage flights and bookings with database integration.

## 🚀 Features
- Add Flights
- Show Flights
- Book Tickets
- Cancel Booking
- User Login System
- View All Bookings
- View User Bookings
- Total Bookings Count

## 🛠️ Tech Stack
- Java (JDBC)
- MySQL
- Eclipse IDE
